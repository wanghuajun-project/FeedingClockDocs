  • App Group: 메인 앱, 위젯, Apple Watch 간 카운트다운 스냅샷 공유(귀하의 기기에서만 처리)
  • Apple Watch: 페어링된 iPhone과 Watch 간 WatchConnectivity로 동기화
  • iCloud Drive(선택, Premium): 수유 데이터를 JSON 형식으로 귀하의 Apple ID iCloud Drive에 저장(Apple이 호스팅)
  • 가져오기 / 보내기: JSON 또는 PDF는 시스템 공유 시트를 통해 귀하가 직접 전송. 당사는 사본을 받지 않습니다

4. 제3자 및 Apple 서비스

  • Apple StoreKit: 구독 및 일회성 구매는 Apple이 처리. 당사는 거래 유효 여부만 수신하며 결제 카드 정보는 처리하지 않습니다
  • 로컬 알림: 수유 알림은 기기에서 로컬로 예약. 제3자 원격 푸시는 사용하지 않습니다
  • 광고·분석 SDK 없음: 제3자 광고 또는 행동 추적 SDK를 사용하지 않습니다
  • 추적 없음: IDFA를 사용하지 않으며, 앱 간 추적을 하지 않습니다

5. 구독, 자동 갱신 및 취소

본 앱은 무료로 이용할 수 있으며, 선택적으로 BottleNudge 프리미엄을 구매할 수 있습니다:

  • 월간: 매월 자동 갱신
  • 연간: 매년 자동 갱신. 7일 무료 체험이 제공될 수 있습니다(App Store 표시 기준)
  • 평생 구매: 일회성 결제. 구독이 아니며 자동 갱신되지 않습니다

가격은 App Store에 귀하 지역 통화로 표시되는 가격을 따릅니다(예: ¥8 / ¥38 / ¥88 또는 $3.99 / $20.99 / $59.99 등. 실제 가격은 구매 시 표시가 우선됩니다).

자동 갱신 안내:

  • 구매 확인 후 Apple ID 계정으로 결제됩니다
  • 현재 결제 기간 종료 24시간 전까지 취소하지 않으면 구독이 자동 갱신됩니다
  • 갱신 요금은 현재 기간 종료 24시간 이내에 계정에서 청구됩니다
  • 무료 체험 기간 미사용분은 유료 구독으로 전환 시 소멸합니다(Apple 규정에 따름)
  • 설정 → Apple ID → 구독에서 관리 또는 취소할 수 있습니다
  • 앱 내 「구매 복원」으로 구매한 권한을 복원할 수 있습니다

Apple 표준 사용권 계약(EULA): Apple Media Services Terms
