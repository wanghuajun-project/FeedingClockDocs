# FeedingClockDocs

NourishBaby（配方奶喂养专家）App 的 **隐私政策** 与 **免责声明** 公开页面，供 App Store 审核与应用内链接使用。

## 页面

| 文件 | 公开 URL |
|------|----------|
| [privacy-policy.html](./privacy-policy.html) | `https://wanghuajun-project.github.io/FeedingClockDocs/privacy-policy.html` |
| [disclaimer.html](./disclaimer.html) | `https://wanghuajun-project.github.io/FeedingClockDocs/disclaimer.html` |

## GitHub Pages 部署

1. 本仓库 **Settings → Pages**
2. **Source**：Deploy from a branch
3. **Branch**：`main`，文件夹选 **`/ (root)`**
4. 等待部署后浏览器验证上述 URL 可访问
